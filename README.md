# E-commerce Batch ELT Pipeline

Proyecto de portafolio de ingeniería de datos con enfoque profesional, orientado a simular un pipeline batch ELT de e-commerce.

## Objetivo
Construir un pipeline que:
- ingiera datos de e-commerce
- los cargue a una capa raw en PostgreSQL
- los transforme con dbt
- genere modelos analíticos de negocio
- se orqueste con Airflow
- incluya pruebas, documentación y CI básico

## Stack base
- Python
- PostgreSQL
- Docker / Docker Compose
- Airflow
- dbt
- GitHub Actions
